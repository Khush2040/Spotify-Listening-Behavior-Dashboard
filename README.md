# 🎵 Spotify Listening Behavior Dashboard | Tableau

## 📌 Project Overview

The **Spotify Listening Behavior Dashboard** is an interactive Tableau project that analyzes a user's Spotify streaming history to uncover listening patterns, artist preferences, song popularity, skip behavior, and music consumption trends. The dashboard transforms raw streaming data into meaningful insights using interactive visualizations and KPIs.

---

## 🎯 Objective

The objective of this project is to:

- Analyze Spotify listening habits.
- Identify top artists and most played songs.
- Understand listening trends over time.
- Analyze skip behavior.
- Discover peak listening hours.
- Compare new versus repeat artist listening patterns.
